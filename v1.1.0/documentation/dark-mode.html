Dark Mode

Materia boasts a sophisticated user interface with a sleek dark mode option that is designed to alleviate eye strain and enhance the user experience.

Activating Dark Mode

You can activate Materia’s sleek dark mode user interface by clicking on the toggle button (with a Sun/Crescent icon) located in the right side of the page. Materia utilizes the local storage to retain the user’s preference for dark mode. Consequently, the next time the user accesses the page, dark mode will be enabled automatically.
